## About the Project

A lightweight, privacy-focused web application that parses Excel files (`.xlsx`) and generates a Google Calendar-compatible file (CSV/iCal) for seamless bulk event uploading. Built with a semantic, accessible web interface, the app is fully capable of running **completely offline**.

---

## Features

* **Offline First:** Process your data locally without worrying about sensitive calendar information leaving your machine.
* **Semantic Web UI:** Built using accessible, clean HTML5 components for an intuitive user experience.
* **Smart Parsing:** Automatically extracts dates, times, titles, and descriptions from uploaded spreadsheets.
* **Google Calendar Ready:** Generates output formats tailored exactly to Google Calendar's import specifications.

---

## The Accessibility First Interface

The web form is built with accessibility and user experience in mind, utilizing modern semantic tags:
* `<form>` & `<fieldset>`: Groups related configuration settings logically.
* `<legend>`: Provides clear captions for form sections.
* `<details>` & `<summary>`: Keeps advanced options tucked away cleanly unless needed.
* `<select>` & `<button>`: Delivers crisp, native interactive elements.
* `<small>`: Offers contextual helper text for file formatting constraints.

## How to Run Locally
Because the application is compiled into a standalone executable, **no installation or Python setup is required.**

1. Head over to the [Releases](../../releases) page of this repository.
2. Download the latest version of `ExcelToCalendar.exe` (or the zipped package).
3. Double-click the `.exe` file to launch the application.
4. Your browser will automatically open to the local web interface, or you can navigate to the local address provided in the terminal window.

*Note: Since the app runs completely offline, your Excel data and calendar details never leave your local machine.*

📅 How to Import into Google Calendar

    Open Google Calendar on your desktop.

    In the top right, click the Settings Gear ⚙️ -> Settings.

    In the left-hand menu, click Import & export.

    Click Select file from your computer and choose the file generated by this app.

    Choose which calendar to add the events to, and click Import.
